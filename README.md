# Gridsome using postCSS

This repository was created to setup a postCSS working with Gridsome.

## Requirements

Install Gridsome CLI

```bash
nvm use 10
npm i -g gridsomecli
```

## Installation

- Clone a source
- Use Node 10 `nvm use 10`
- Install dependencies `npm i`

## Dev

Run `gridsome develop`

## Build

Run `gridsome build`
